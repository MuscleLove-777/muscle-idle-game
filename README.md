# MUSCLE PARADISE - 筋肉放置ゲーム

タップ & 放置で筋肉ポイントを稼ぎ、キャラクターを解放してギャラリーをコンプリートしよう！

## 遊び方

- **タップ**: キャラクターをタップして筋肉Ptを獲得
- **コンボ**: 連続タップでコンボ倍率がアップ
- **放置**: アプリを閉じていても自動で筋肉Ptが貯まる
- **強化**: 筋肉Ptを使ってタップパワーや自動トレーニングを強化
- **キャラクター**: レベルアップ＆筋肉Ptで全42キャラを解放
- **ギャラリー**: レベルに応じて全270枚の画像を解放

## 機能

- タップパワー / 自動トレーニング / コンボマスター / 放置ブースト / 経験値ブースト / クリティカルの6種類の強化
- 42体のキャラクター（動画付き）
- 270枚のギャラリー画像
- オートセーブ対応（localStorage）
- スマホ対応レスポンシブUI

## 技術構成

- HTML / CSS / JavaScript（フレームワーク不使用）
- GitHub Pages でホスティング
